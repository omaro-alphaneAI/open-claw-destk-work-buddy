# 小七素材需求 Brief

这一版桌宠已经有运行中的占位素材，但要把视觉收稳，还需要一套统一输出标准。下面这份 brief 用来约束头像、立绘和托盘图标，不再靠临时拼。

## 1. 头像

- 用途：聊天抽屉、工作台、后续多会话列表
- 风格：小只、利落、偏西海岸松弛感的小黑猫
- 基础状态：待命 / 思考 / 完成 / 警示
- 输出格式：`SVG` 主稿 + `PNG @2x`
- 推荐尺寸：`256 x 256`
- 背景：透明
- 要求：
  - 脸部轮廓要在 24px 级别仍可辨识
  - 警示态不能做成“发怒”，而是“我先拦一下”的警觉感
  - 完成态要偏干净利落，不要过度卖萌

## 2. 立绘

- 用途：桌面本体、工作台头图、宣传图
- 风格：轻盈、敏捷、黑猫，带一点街头和 West Coast 气质
- 视角：侧身 3/4 为主，方便延展到动作帧
- 输出格式：`SVG` 分层主稿 + `PNG`
- 推荐尺寸：主立绘 `1024 x 1024`
- 需要覆盖：
  - 常驻待机站姿
  - 轻思考姿态
  - 完成后收尾姿态
  - 警示 / 抬爪拦一下姿态
- 要求：
  - 四肢和尾巴层最好可拆，方便后续继续动画化
  - 黑色主体要留层次，不能糊成一团纯黑
  - 眼神表达优先于夸张肢体

## 3. 托盘图标

- 用途：macOS tray / menu bar
- 风格：极简、单色优先、辨识度高
- 输出格式：`Template PNG` + `PDF/SVG`
- 推荐尺寸：
  - `16 x 16`
  - `18 x 18`
  - `32 x 32`
- 要求：
  - 以猫头或抬耳轮廓为主，不塞细节
  - 深浅模式下都能稳
  - 高亮态和默认态都要测试

## 4. 分身猫延展

- 当前已有分身：主猫 / 补丁猫 / 侦查猫 / 巡航猫
- 建议保持同一骨架，只改：
  - 领结 / 耳标 / 小挂件
  - 辅助色
  - 眼神和道具
- 不要把四只分身画成四个无关 IP

## 5. 交付要求

- 文件命名稳定，不要混乱版本号
- 每张图附一个 1 句用途说明
- 统一放在 `src/assets/`
- 需要同时给设计稿和导出稿
